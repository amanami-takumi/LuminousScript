# LuminasScript アップデート完了報告

## 実装した変更内容

### 1. ✅ 立ち絵の大きさを3倍程度に拡大
- キャラクタースプライトのサイズを `width: 30%` → `width: 40%` に変更
- 高さを `height: 80%` → `height: 95%` に変更
- より大きく、迫力のある立ち絵表示を実現

### 2. ✅ 会話履歴機能の追加
- 新しい「会話履歴」ボタン(📜)を追加
- すべての会話を自動的に記録
- 履歴画面で過去の会話を遡って確認可能
- 話者名とテキストを分けて表示
- スクロール可能なリスト形式

### 3. ✅ クリック制限時間のゲージ表示
- テキストボックスの右下に進行状況ゲージを追加
- 500ミリ秒のクリック遅延を視覚的に表示
- ゲージが満タンになるとクリック可能に
- グラデーションカラーで美しく表示

### 4. ✅ 自動進行機能の追加
- 新しい「自動」ボタン(▶)を追加
- ボタンをクリックで自動モードのON/OFF切り替え
- 自動モード中は3秒ごとに自動的に次のシーンへ進む
- アクティブ時はボタンの色が変わる

### 5. ✅ ローディング画面の追加
- ゲーム開始時に「ローディング中...」画面を表示
- 回転するスピナーアニメーション
- 1秒後にフェードアウトしてゲーム画面へ
- 美しいグラデーション背景

### 6. ✅ config.yml対応
- `input/config.yml`からカスタマイズ設定を読み込み
- 以下の項目をカスタマイズ可能:
  - ゲームタイトル (`adv_title`)
  - サブタイトル (`adv_sub_title`)
  - タイトル背景画像 (`title_bg_image`)
  - 制作者名 (`creator_name`)
  - テーマカラー (`theme_color`, `sub_color`)
  - フォント (`text_font_importURL`)
  - SNSリンク (X, VRChat, Fediverse, Web, BOOTH)
  - ファビコン (`favicon_url`)

### 7. ✅ クレジット表示の追加
- 「Generated by Luminous Script」を必ず表示
- Apache License 2.0のライセンス情報を表示
- 「ライセンスはスクリプトにのみ適用」と明記
- config.ymlの設定に関わらず常に表示
- 制作者情報とSNSリンクも併せて表示

## 技術的な改善点

### 新しい依存関係
- `PyYAML`: config.ymlの読み込みに使用
- インストール: `pip install pyyaml`

### コード構造の改善
- `load_config()` メソッドを追加してconfig.ymlを読み込み
- 会話履歴管理機能を追加
- 自動モードのタイマー管理を追加
- クリック遅延のゲージアニメーション実装

### UI/UXの向上
- より大きな立ち絵で没入感アップ
- 会話履歴で物語を振り返り可能
- クリック可能タイミングの視覚的フィードバック
- 自動モードで手軽にプレイ可能
- ローディング画面でプロフェッショナルな印象

## 使用方法

1. `input/config.yml`を編集してゲームをカスタマイズ
2. `python luminas_script.py`を実行
3. `output/game.html`が生成される
4. ブラウザで開いてプレイ
